#+TITLE: Intellij Idea


* 快捷方式

最常用，务必要记住系列:

| 按键          | 作用                     |
|---------------+--------------------------|
| ~Alt-Enter~   | 万能补全                 |
| ~Alt-Insert~  | 生成构造器、get/seter 等 |
| ~Shift+Shift~ | 快速查找                 |
| ~Shift-F6~    | 重命名                   |
| ~Ctl-Alt-L~   | 格式化代码               |
