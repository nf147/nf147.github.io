1 图示

MySQL InnoDB 引擎默认的隔离级别是 RR(Repeatable Read): 

show variables like 'tx_is%';

-- 更改
set global tx_isolation='read-committed';
set global tx_isolation='repeatable-read';

Oracle/PostgreSQL 等数据库的默认隔离级别是 RC(Read Committed)。
